You will need game pads to play this game. (Xbox or Playstation)

CONTROLS:

Steer: Joysticks

Accelerate: Right Trigger

Reverse, Brakes: Left Trigger

Boost: Right Bumper


CONTRIBUTIONS:

For this game I worked mainly on developing a car controller. The script can be found at Unity Project/SpiralBoiz/Assets/Scripts/CarController.cs
